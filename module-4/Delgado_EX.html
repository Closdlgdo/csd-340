5 Things I've Learned Since the Course Started

  • HTML provides structure to the things that appear on a website, such as images, text, or videos.
  • Only content inside the opening and closing body tags can be displayed to the screen.
  • There can be multiple levels of nesting. The relationship between elements and their ancestor and descendent elements is known as hierarchy.
  • There are six different heading elements and can be used for a variety of purposes, like titling sections or other forms of content.
  • Divs allow us to group HTML elements to apply the same styles for all HTML elements inside.
